# BNI BINGO 紅黃綠燈平台 — 團隊交接

## 正式網址

<https://bni-traffic-light.bingo-win.workers.dev/>

請只分享以上網址。團隊不需要使用 `/v2.html`、preview link 或 release query parameter。

## 平台用途

### 會員

- 使用會員共用密碼登入
- 搜尋自己的姓名
- 查看最新燈號及 Excel 正式分數
- 查看個人化綠燈行動建議
- 查看月份表現變化
- 下載個人分享圖

### LT

- 使用 LT 專用密碼登入
- 上載及核對每月 Traffic Excel
- 發布最新／歷史月份或取代錯誤月份
- 查看會員趨勢及月份紀錄
- 批量下載改善圖卡 ZIP

## 改善圖下載

預設下載：

- 黃燈
- 紅燈
- 黑燈

綠燈會員預設不包括在改善圖內。LT 亦可選擇：

- 全部會員
- 只下載黃燈
- 紅燈及黑燈

## 正式分數及建議的分別

- 「Excel 正式分數」是正式結果。
- 「綠燈行動建議」是按原始數據及 70 分綠燈門檻計算的行動路徑。
- 建議不會修改正式分數。
- 實際下月結果以新上載 Excel 為準。

## 權限

- 會員密碼可提供給 BINGO 會員。
- LT 密碼只可私下提供給獲授權 LT。
- 不要在公開群組、社交媒體或文件寫出 LT 密碼。

## LT 每月最簡流程

1. 匯出原始 Traffic Excel。
2. 登入 LT 管理。
3. 選擇 Excel。
4. 核對月份、會員數、七項正式分數及預覽。
5. 下載核對 CSV。
6. 正式發布。
7. 抽查 3 位會員。
8. 下載黃燈或以下 ZIP 並核對圖卡。

完整 SOP 見 [WORKFLOW.md](WORKFLOW.md)。

## 遇到問題

請記錄以下資料再交給技術維護人：

- 發生時間
- 使用手機或電腦
- 瀏覽器名稱
- 所在頁面
- 做了哪些步驟
- 畫面截圖
- 錯誤訊息原文
- 是否只有自己遇到

不要把 LT 密碼、session token 或完整 Excel 放到公開群組。原始 Excel 由 LT 負責人私下保存，平台不會代為備份。
